# Star MSA - Algoritmo de Alineamiento Múltiple en Estrella

Este programa implementa el algoritmo de alineamiento múltiple en estrella (Star MSA) para secuencias de ADN.

## Compilación

Para compilar el código fuente `star.cpp`, usa el siguiente comando en la terminal:

```bash
g++ star.cpp -o star
```

Esto generará un archivo ejecutable llamado `star`.

---

## Ejecución

Para ejecutar el programa y leer las secuencias desde el archivo, utiliza:

```bash
./star < BRCA1.txt
```
## Archivos incluidos

- `star.cpp` → Código fuente en C++.
- `BRCA1.txt` → Archivo de entrada con secuencias.
- `README.md` → Instrucciones de compilación y uso.
